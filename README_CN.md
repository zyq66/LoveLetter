# 情书 💌

**中文** | [English](README.md)

> 专属于两个人的私密空间 — 记录每一个珍贵瞬间，让爱意每天都有迹可循。

---

## 什么是情书？

**情书**是一款专为情侣打造的私密移动应用。它不是社交平台，没有陌生人，没有公开的动态——只属于你们两个人。从在一起的第一天到每一个纪念日，从日常的小情绪到深夜想说的话，情书让感情有温度地被记录下来。

---

## 为什么选择情书？

大多数通讯软件是为世界设计的，情书只为两个人设计。

没有粉丝，没有点赞，没有公开的朋友圈——只有一个安静、私密的空间，让你们可以完全地陪伴彼此。每一张照片、每一条心情、每一个微小的瞬间，都只属于你们。

---

## 功能亮点

**💬 私密聊天** — 支持情绪标签、图片分享、表情回应，以及 AI 情书生成——当你不知道说什么的时候，它帮你把爱说出口。

**📔 恋爱日记** — 以时间线方式记录两人的共同故事，每条记忆都可配图、写注记、打上心情贴纸。

**📸 共享相册** — 实时同步的私密相册，两人的照片永远整齐地在一起。

**💝 专属档案** — 记录对方的生日、血型、MBTI、爱吃什么、有什么习惯和爱好，一份只属于你的深度了解。

**🗓 纪念日守护** — 在一起的每一天都被计数，重要的日子提前提醒，再也不会忘记。

**🤖 AI 陪伴** — 由 DeepSeek 驱动，生成情书、提供话题灵感、分析心情、送上暖心祝福——让两人随时都有话聊，有爱可表达。

## 技术架构

| | |
|---|---|
| 框架 | React Native + Expo SDK 54（TypeScript）|
| 后端 | 腾讯云开发 — 数据库 · 存储 · 云函数 |
| AI | DeepSeek API（云函数代理） |
| 构建 | EAS Build（Android APK）|

> 所有运行时网络请求均经由腾讯云国内节点，无境外依赖，在中国大陆稳定运行。

---

## 快速开始

```bash
npm install
npm start        # Expo 开发服务器
npm run build    # EAS 打包 Android APK
```

**必要配置：**
- `src/config/cloudbase.ts` — 填入 TCB 环境 ID
- `src/services/storage.ts` — 填入 TCB 存储域名
- `functions/ai-proxy/` — 部署至 TCB 控制台，设置 `DEEPSEEK_API_KEY` 环境变量，超时改为 20s
